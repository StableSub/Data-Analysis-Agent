# 참고 자료

이 디렉터리는 발표, 제안서, 외부 입력처럼 제품과 구현 판단에 참고할 수 있는 자료를 보관한다.
여기 있는 문서는 현재 runtime 계약이나 검증 기준을 직접 소유하지 않는다. 현재 동작은 `docs/architecture/`와 `docs/system/`, 제품 범위는 `docs/product/`, 개발·검증 명령은 루트 `AGENTS.md`와 `README.md`를 기준으로 확인한다.

## 포함 자료

| 자료 | 역할 | 비고 |
|---|---|---|
| `산학캡스톤_프로젝트_제안서.pdf` | 프로젝트 제안서 원문 | 외부 입력 자료 |

## 사용 기준

- 발표 자료나 제안서의 배경을 확인할 때 이 디렉터리를 읽는다.
- 현재 구현 상태를 판단할 때는 [현재 구현 기준선](../product/current-state-baseline.md)과 [아키텍처 문서](../architecture/README.md)를 우선한다.
- 외부 자료와 코드가 다르면 현재 코드와 architecture 문서를 기준으로 최신 설명을 작성한다.
