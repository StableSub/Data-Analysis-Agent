# 시스템

시스템 전체 흐름과 기술 구조를 설명하는 문서를 모아두는 디렉터리다.

이 디렉터리는 구조/API 진입점을 설명한다. 질문의 시간 순서와 state 계약은 각각 `../request-lifecycle.md`, `../shared-state.md`를 기준으로 보고, 여기서는 중복 설명하지 않는다.

- `backend-structure.md`: 백엔드 구조
- `frontend-structure.md`: 프론트엔드 구조
- `api-spec.md`: API 개요 및 명세

실행 흐름은 중복 문서로 분리하지 않고 `../request-lifecycle.md`를 기준으로 본다.
